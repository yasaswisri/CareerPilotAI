# 🚀 CareerPilot AI – Your Intelligent Career Assistant

An AI-powered career guidance platform built using **Streamlit** and **Google Gemini AI** that helps students and job seekers improve their career journey through resume analysis, interview preparation, career guidance, and personalized learning roadmaps.

---

## 📖 Project Overview

CareerPilot AI is an intelligent web application designed to simplify career planning. It combines Artificial Intelligence and Generative AI to provide users with personalized career recommendations, resume analysis, interview coaching, and learning resources—all from one platform.

The project was developed as part of the **Kaggle AI Agents Intensive Vibe Coding Capstone Project**.

---

- 📄 AI Resume Analyzer

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/yasaswisri/CareerPilotAI.git
```

### Move into Project

```bash
cd CareerPilotAI
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

Windows

```bash
venv\Scripts\activate
```

Linux/Mac

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Create Environment File

Create a `.env` file.

```env
GEMINI_API_KEY=YOUR_API_KEY
SMTP_EMAIL=YOUR_EMAIL
SMTP_PASSWORD=YOUR_PASSWORD
```

### Run Application

```bash
streamlit run app.py
```
